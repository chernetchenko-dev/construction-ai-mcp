# Скилл: Создание семейств Revit по BIM-стандарту 2.0

## Цель скилла
Помочь разработать, проверить, задокументировать или проаудировать через MCP семейство Revit в соответствии с требованиями BIM-стандарта 2.0 и ФОП 2021.

---

## Шаг 1 — Определить задачу

Уточни у пользователя:
1. **Что нужно сделать?** (создать / проверить / составить ТЗ / чек-лист / аудит через MCP)
2. **Категория Revit** — двери, окна, оборудование, воздуховоды, фитинги и т.д.
3. **Дисциплина** — АР, КР, ОВ, ВК, ЭОМ, ТХ
4. **Вид семейства** — системное / загружаемое / вложенное / общее вложенное

---

### Базовые (обязательные для всех)

| 1 | Масштаб 1:1, метрическая система | 4.1.6, 4.2.1 |

| 8 | Нет импортированной геометрии (DWG/SAT) | 7.9 |
| 9 | Имена параметров: ADSK_ только shared, остальные кириллица без префикса | 4.8.5 |

## Шаг 4 — Общие параметры ФОП 2021

**Критически важно:** ADSK_-параметры должны быть **общими (IsShared = True)** из ФОП. Локально добавленные ADSK_-параметры — ошибка: спецификации не агрегируют данные, IFC-экспорт некорректен.

### Обязательные параметры (Группа 01 — Общие):
| Параметр ADSK | Тип | Назначение |
|---------------|-----|-----------|
| ADSK_Наименование | TEXT | Полное наименование для спецификации |
| ADSK_Наименование краткое | TEXT | Краткое для марок |
| ADSK_Марка | TEXT | Марка изделия |
| ADSK_Код изделия | TEXT | Артикул / код по каталогу |
| ADSK_Позиция | TEXT | Позиция в спецификации |
| ADSK_Количество | NUMBER | Количество |
| ADSK_Масса | NUMBER | Масса единицы |
| ADSK_Обозначение | TEXT | Обозначение документа |
| ADSK_Примечание | TEXT | Примечание |
| ADSK_Завод-изготовитель | TEXT | Производитель |
| ADSK_Единица измерения | TEXT | Ед.изм. |
| ADSK_Техническая характеристика | TEXT | Технические характеристики |

### Параметры размеров (Группа 10):
| Параметр ADSK | Тип | Назначение |
|---------------|-----|-----------|
| ADSK_Размер_Ширина | LENGTH | Ширина |
| ADSK_Размер_Высота | LENGTH | Высота |
| ADSK_Размер_Длина | LENGTH | Длина |
| ADSK_Размер_Толщина | LENGTH | Толщина |
| ADSK_Размер_Диаметр | LENGTH | Диаметр (круглые) |

Полный перечень 323 параметров → `references/fop-params.md`

## Шаг 5 — Коннекторы MEP и сортамент по ВСН 353-86

**Правило:** размер коннектора воздуховода округляется до ближайшего стандартного типоразмера по ВСН 353-86. Произвольные размеры патрубков не допускаются.

### Прямоугольные воздуховоды (ВСН 353-86):
```
100×150, 150×150, 150×250, 150×300,
250×250, 250×300, 250×400, 250×500,
400×400, 400×500, 400×600, 400×800,
500×500, 500×600, 500×800, 500×1000,
600×600, 600×800, 600×1000, 600×1250,
800×800, 800×1000, 800×1250, 800×1600,
1000×1000, 1000×1250, 1000×1600, 1000×2000,
1250×1250, 1250×1600, 1250×2000,
1600×1600, 1600×2000 мм
```

### Круглые воздуховоды (ВСН 353-86, п.1.1):
```
100, 125, 160, 200, 250, 315, 355, 400, 450, 500,
560, 630, 710, 800, 900, 1000, 1120, 1250, 1400,
1600, 1800, 2000 мм
```

### Примеры округления:
| Патрубок оборудования | Стандарт ВСН | Комментарий |
|-----------------------|-------------|-------------|
| 810×710 мм | **800×800** мм | Ближайший снизу |
| 450×450 мм | **400×500** мм | Или 500×500 с запасом |
| 320×220 мм | **250×300** мм | |
| 480 мм (круглый) | **500** мм | Ближайший диаметр |

### Настройка коннекторов:
- Тип системы: Воздуховод / Трубопровод / Кабелепровод
- Направление потока: Вход / Выход / Двунаправленный
- Размер привязать к параметрам семейства (не хардкодить числа)
- ADSK_Размер_Ширина и Высота синхронизировать с размером коннектора

## Шаг 9 — Аудит через MCP (Revit открыт)

Если семейство открыто в Revit и доступен `Revit Connector:execute_revit_code` — проводить аудит скриптами.

### ⚠️ Обязательные правила кодировки (IronPython / pyRevit):
- `print(str.encode('utf-8'))` — для **любой** кириллицы в `print()`
- В `return`-значениях кириллица без encode — нормально
- Никогда: `print(u"кириллица")` → UnicodeEncodeError

### Скрипт 1: Базовый аудит

```python
from pyrevit import revit, DB
doc = revit.doc
print("IsFamilyDocument:", doc.IsFamilyDocument)
fam_mgr = doc.FamilyManager
fam_cat = doc.OwnerFamily.FamilyCategory
print("Категория:", fam_cat.Name.encode('utf-8'))
types = list(fam_mgr.Types)
print("Типоразмеров:", len(types))
for t in types:
    print("  ", t.Name.encode('utf-8'))
params = list(fam_mgr.Parameters)
adsk_shared = [p for p in params if p.IsShared and p.Definition.Name.startswith("ADSK_")]
adsk_local  = [p for p in params if not p.IsShared and p.Definition.Name.startswith("ADSK_")]
user_other  = [p for p in params if not p.Definition.Name.startswith("ADSK_") and not p.IsShared]
print("ADSK_ shared (правильно):", len(adsk_shared))
print("ADSK_ локальные (ОШИБКА):", len(adsk_local))
for p in adsk_local:
    print("  ИСПРАВИТЬ:", p.Definition.Name.encode('utf-8'))
print("Прочие пользовательские:", len(user_other))
for p in user_other:
    print("  ", p.Definition.Name.encode('utf-8'))
```

### Скрипт 2: Подкатегории, коннекторы, импорт, LOD

```python
from pyrevit import revit, DB
doc = revit.doc
fam_cat = doc.OwnerFamily.FamilyCategory
print("Подкатегорий:", fam_cat.SubCategories.Size)
for sc in fam_cat.SubCategories:
    print("  ", sc.Name.encode('utf-8'))
conn_elems = list(DB.FilteredElementCollector(doc).OfClass(DB.ConnectorElement).ToElements())
print("ConnectorElement:", len(conn_elems))
imports = list(DB.FilteredElementCollector(doc).OfClass(DB.ImportInstance).ToElements())
print("ImportInstance (должно быть 0):", len(imports))
params = list(doc.FamilyManager.Parameters)
mat_params = []
for p in params:
    try:
        if str(p.Definition.ParameterType) == "Material":
            mat_params.append(p.Definition.Name)
    except: pass
print("Параметров-материалов:", len(mat_params))
views = list(DB.FilteredElementCollector(doc).OfClass(DB.View).ToElements())
lods = set()
for v in views:
    try:
        dl = v.get_Parameter(DB.BuiltInParameter.VIEW_DETAIL_LEVEL)
        if dl: lods.add(dl.AsInteger())
    except: pass
lod_names = {1: "Грубый", 2: "Средний", 3: "Точный"}
print("LOD виды:", [lod_names.get(l,l) for l in sorted(lods)])
```

### Скрипт 3: Проверка ADSK_Размер_* по типоразмерам

```python
from pyrevit import revit, DB
doc = revit.doc
fam_mgr = doc.FamilyManager
param_index = {p.Definition.Name: p for p in fam_mgr.Parameters}
adsk_params = [n for n in param_index if n.startswith("ADSK_Размер_")]
print("ADSK_Размер_ параметры:", len(adsk_params))
for t in fam_mgr.Types:
    fam_mgr.CurrentType = t
    print("Тип:", t.Name.encode('utf-8'))
    for pname in adsk_params:
        p = param_index.get(pname)
        if p:
            try:
                val_mm = t.AsDouble(p) * 304.8
                print("  {} = {:.0f} mm".format(pname.encode('utf-8'), val_mm))
            except: pass
```

### Скрипт 4: Установка ADSK_Размер_Ширина/Высота (транзакция)

```python
from pyrevit import revit, DB
doc = revit.doc
fam_mgr = doc.FamilyManager
param_index = {p.Definition.Name: p for p in fam_mgr.Parameters}

# Задать вручную: {подстрока_имени_типа: (ширина_мм, высота_мм)}
SIZE_MAP = {
    u"П3.1": (300.0, 300.0),
    u"П6.1": (300.0, 300.0),
    u"03":   (800.0, 700.0),
}

adsk_w = param_index.get(u"ADSK_Размер_Ширина")
adsk_h = param_index.get(u"ADSK_Размер_Высота")
if not adsk_w or not adsk_h:
    print("ERROR: ADSK_Размер_Ширина или Высота не найдены")
else:
    with revit.Transaction("Set ADSK sizes"):
        for t in fam_mgr.Types:
            fam_mgr.CurrentType = t
            tname = t.Name
            sz = next((v for k, v in SIZE_MAP.items() if k in tname), None)
            if sz:
                fam_mgr.Set(adsk_w, sz[0] / 304.8)
                fam_mgr.Set(adsk_h, sz[1] / 304.8)
                print("OK [{}] W={:.0f} H={:.0f}".format(tname.encode('utf-8'), sz[0], sz[1]))
            else:
                print("SKIP:", tname.encode('utf-8'))
```

## Типичные ошибки и исправления

| Ошибка | Симптом | Исправление |
|--------|---------|-------------|
| ADSK_-параметры локальные | `IsShared == False` для ADSK_ | Удалить → подключить ФОП → добавить как shared |
| Размер коннектора ≠ ВСН | Система не подключается | Округлить по сортаменту ВСН 353-86 |
| Кириллица в print() | UnicodeEncodeError в MCP | `.encode('utf-8')` для строк в print() |
| Нет каталога типоразмеров | >6 типов, нет .txt каталога | Создать каталог рядом с .rfa |
| Импортированная геометрия | ImportInstance > 0 | Удалить: Управление → Удалить импорт |
| ADSK_Размер_* = 0 или 1 фут | Не заполнены | Скрипт 4 (установка через транзакцию) |
